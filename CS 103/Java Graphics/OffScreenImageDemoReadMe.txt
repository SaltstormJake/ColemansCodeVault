1. Create a new folder in the OffScreenImageDemo and call it Images.
2. Copy into this folder any JPG image. It is recommended that the size of the image be approximately 250 x 400 pixels.
3. Look for the place in the code below where the image Duke.jpg is being loaded and substitute the name of your image.
4. Save - Compile - Run. Study the comments and study the grahics object at the Java API website (link is found on the previous page).
